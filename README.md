# discord_transfer

## Requirements

Node 12+

## Run

Create `config/secrets.js` and set tokens

```
module.exports = {
  from_token: "<BOT_A_TOKEN>",
  to_token: "<BOT_B_TOKEN>"
}
```

Install dependencies & Run bots

```shell
$ npm install
$ npm run start
```
